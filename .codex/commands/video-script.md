Контекст для модели:
- Аргумент `$ARGUMENTS` может содержать: только тему («бутылочное горлышко в закупках»), тему + детали кейса, или тему + дату/время публикации (express mode: «про тендеры в нефтянке, публикация 2026-05-01 14:00»).

- Дальше следуй workflow из skill SKILL.md: tone-of-voice → структура (хук/тезис/теория/кейс/резюме) → утверждение → телесуфлёр-текст → сохранить DOCX в `.tmp/teleprompter/`.
